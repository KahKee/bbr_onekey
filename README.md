# bbr_onekey
修改秋水的bbr

## 一键命令（换行空两格）
wget --no-check-certificate -O /opt/bbr.sh https://cdn.jsdelivr.net/gh/KahKee/bbr_onekey@raw/main/bbr.sh 
chmod 755 /opt/bbr.sh  
/opt/bbr.sh  

## 修改内容
由 [elrepo](http://elrepo.org/tiki/tiki-index.php) 控制安装centos7内核，默认安装稳定主线内核kernel-ml
